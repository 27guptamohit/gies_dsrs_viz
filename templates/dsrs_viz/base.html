About the Project
August 18, 2020

The project consists of three parts.

  • D3 JS: Tab containing visualizations of D3 based libraries
  • React JS: Tab containing visualizations of React based libraries
  • React-D3 JS: Hybrid libraries based on both D3 and React
  • Admin : Admin controlled applications to control contents of above three project parts
